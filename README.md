## 🌡️ Conversor de Temperaturas
Um projeto simples em python de conversão térmica que transforma graus Celsius em Fahrenheit e Kelvin simultaneamente.

### Conceitos aplicados:
* **Entrada e Saída (I/O):** Interação direta com o usuário via terminal.
* **Casting:** Conversão de dados de entrada para `float` para precisão nos cálculos.
* **Fórmulas Matemáticas:** Implementação de lógicas de conversão física.
* **Formatação de Strings:** Exibição clara e organizada das unidades de medida.
